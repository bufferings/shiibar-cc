## タスク

### T1: RowSymbolView の描画差し替え

- **idle**: `Text("✻")`(U+273B。emoji 化を防ぐため text presentation を保証 — 実機で
  カラー絵文字にならないことを確認)、`.primary` の 50% 透明度、静止。
  グリフサイズはビュー枠 20px に対して 15〜16px 相当から実機調整
- **waiting**: 吹き出しの輪郭(角丸矩形 + 左下の尻尾、線幅 ~1.7/20 スケール・塗りなし)+
  中央に太い `!`。パスのジオメトリは menubar-design.html のモック SVG の値を出発点にする
- **working**: グリフ循環スピナー。フレーム = `·` `✢` `✳` `✶` `✻` `✽`(各グリフに
  U+FE0E を付けて text presentation を保証)、フレーム位置 =
  `round((1−cos(2π·経過秒/2))/2 × 5)` — **周期 2 秒のコサインイージング**(両端に留まり
  中間を素早く通過する。等間隔ステップにしない。claude CLI 2.1.204 の実装から読み取った実式)。
  **壁時計駆動の TimelineView**(50ms 刻み・`spinning` ゲート = 開いている間だけ・
  複数行が同位相、はそのまま維持)。濃度 100%・サイズは idle の `✻` と同じ枠。
  グリフごとの光学サイズ差(`✳` `✶` は字面が小さめ)が実機で気になる場合は、
  フレーム別のスケール補正を入れてよい(実機調整)
- **バッジ**: 現行実装のまま(位置・サイズ・ハロー変更なし)
- 3 記号の光学的な大きさ(見た目の重さ)がそろうよう、サイズ定数は実機で最終調整する
  前提のコメントを残す(既存の TrayIconMetrics と同様の考え方)
