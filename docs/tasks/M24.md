## 背景とゴール

M22 で行シンボルを本家 TUI の語彙(! / グリフ循環スピナー / 静止 ✻)に刷新した。
本 M はそれを製品全体に波及させる(所有者決定済み):

1. **トレイ**を「窓 + 1 グリフ」文法にする — 左上スロットの中身がロールアップそのもの
2. **アプリアイコン**を「窓タイル + ✻ + 右肩赤バッジ」に刷新(状態記号の縦積み廃止)
3. 紋章のグリフを製品全体で **✳(U+2733)→ ✻(U+273B)** に統一

## タスク

### T1: トレイ(`app/Sources/ShiibarCcApp/TrayIconRenderer.swift` ほか)

20×18 論理単位は不変。**左上スロット(現行の紋章位置・x 6.4 基準)の 1 グリフ**が状態を運ぶ:

- **waiting**: スロットに**太い `!`**(紋章と入れ替え。サイズは紋章の約 1.1 倍・heavy)。
  現行の右下の `!` は廃止
- **working**: スロットの **✻ が瞬く** — RowSymbolView と同じグリフ循環
  (フレーム `·✢✳✶✻✽` 各 U+FE0E 付き、位置 = `round((1−cos(2π·経過秒/2))/2 × 5)`、
  50ms 刻み、§9)。現行の右下の点灯ドットは廃止。
  タイマーは現行どおり **working 表示のときだけ**動かす(仕組みは流用)
- **idle(全員)**: 静止 ✻。全体減光 0.8(現行のまま)
- **エージェントなし・切断**: 静止 ✻。全体減光 0.45(現行のまま)
- **unreviewed バッジ**: 右肩・明色ハロー、現行のまま(別レイヤー合成も現行のまま)
- **紋章サイズ**: 現行 12 相当 → **9.5 相当を出発点**に実機調整。
  実機の Retina メニューバーで ✻ の花弁が潰れる場合は 10.5 まで戻してよい(判読が最優先)
- Core の `Rollup` のロールアップ判定(waiting > working > idle)は無変更。
  点灯ドット用の定数(4 コマ・500ms)が他から参照されなくなる場合は削除してよいが、
  既存テストを壊さないこと(テストが定数自体を検証している場合は完了報告で指摘)
