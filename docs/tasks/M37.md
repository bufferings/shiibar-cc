## タスク

### T1: ⏺ マーカー(§4.6 / conversations-design.html)

- **すべての Claude の発言の先頭**に ⏺(淡い二次色・本文より小さい)。連発時だけの条件分岐はしない
- **ぶら下げ**: マーカーは本文の左のグリフ列に置き、折り返し・2 段落目以降・コードブロックが
  マーカーの下に回り込まない(発言の本文全体が 1 段内側の本文列に入る)
- **グリフ列の揃え**: 帯の `❯` と ⏺ が同じグリフ列、帯の文字と本文が同じ列に揃う
  (conversations-design.html の CSS — グリフ列幅 + 間隔 — を目安に)
- ⏺ は描画後テキストに**含めない**(ヒット計算・畳み境界の入力は不変 — Core の
  `RenderedMessage` は無変更のはず)。状態の意味を持たせない(色分けしない)
